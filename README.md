# Sushi-Restaurant

This is a modern, visually appealing website for a fictional sushi restaurant. The project focuses on user experience, responsive layout, and engaging animations to create a dynamic and attractive interface.

## Features

- **Aesthetic and elegant design**
- **Responsive layout** for different screen sizes
- **GSAP animations** for smooth transitions and interactive elements
- **ScrollTrigger integration** for scroll-based effects
- **Well-structured and semantic HTML**

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **GSAP (GreenSock Animation Platform)**
  - ScrollTrigger plugin

## Project Highlights

- Handcrafted layout inspired by modern sushi restaurants
- Scroll animations that bring elements to life as the user navigates the page
- Emphasis on design quality, including typography, color harmony, and spacing
- Lightweight and optimized for performance
